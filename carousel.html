Build a Carousel library using Vanilla JS (use no external library).

A new carousel should be created by writing:

var sliderEl = document.querySelector(‘.slider’)[0]

var slider = new Slider(sliderEl)


Design the DOM structure for this library and also write the function Slider to implement the same


Features:

  1. Each Slide would take up the width of the parent.

  2. Previous and Next buttons for switching the slides

  3. A normal slide transition on switching slides

  4. Autoplay

  5. Destroy slider from DOM


Evaluation Criteria:

  1. Performance i.e. Animation , memory management

  2. Event management

  3. OOJ concepts
